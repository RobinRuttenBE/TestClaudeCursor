# Covering a Base Pole / Frame with 360 Balloons

> **Source:** Sempertex Europe tutorial video — Alberto Falcone (Brand Ambassador)
> **Difficulty:** Beginner
> **Category:** Frame covering, structural technique, event decoration
> **Products used:** Sempertex 360s (two colors — e.g., yellow and lime green)
> **Video reference:** Balloon tutorial on covering a base pole using 360 balloons

---

## Overview

A quick technique for covering any visible pole, stick, or frame structure using two colors of 360 balloons in an alternating spiral pattern. Essential for event decoration where the structural frame needs to be hidden or turned into a design element. Works on any type of frame — metal poles, PVC pipe, wooden dowels, etc.

---

## Technique: Two-Color Spiral Wrap

### Materials
- 2x Sempertex 360 balloons (different colors)
- A pole or frame to cover

### Steps

1. **Inflate small bubbles** at the nozzle end of both 360s. Only inflate enough to tie the two balloons together — the rest of the 360 remains uninflated and will be used to wrap.

2. **Tie the two 360s together** at the inflated bubble ends. This creates your starting anchor point on the frame.

3. **Start wrapping with color 1.** Begin at the base and wrap the first color (e.g., yellow) around the pole in a spiral, working upward.

4. **Switch to color 2.** When you've covered a section with the first color, follow behind it with the second color (e.g., lime green), wrapping over and covering the first color until you reach where the first color started.

5. **Alternate.** Once the second color catches up to the first, swap positions — continue with the color that was underneath, and the other one follows. This creates an alternating color pattern along the pole.

6. **Continue to the top.** Repeat the alternating pattern all the way up the frame.

### Result
A cleanly covered frame with an alternating two-color spiral pattern that hides the structure completely.

---

## When to Use This Technique

- **Visible frame structures** at events that need to be hidden or decorated
- **Arch bases and columns** where the pole is partially visible between balloon clusters
- **Stage framing** where structural supports need to match the color scheme
- **Any installation** where the frame is not fully covered by balloon clusters and the bare structure would be visible

---

## Tips

- Only inflate small bubbles for tying — the uninflated latex does the wrapping
- Wrap tightly for a clean, uniform look
- Works with any two-color combination — match to the event's color scheme
- Can also be done with a single color for a simpler look
- The 360 size is ideal because the extra length (3" x 60") provides plenty of material to cover longer poles without needing to tie on additional balloons
