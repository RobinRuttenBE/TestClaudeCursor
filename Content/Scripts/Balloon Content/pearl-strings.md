## Overview

Pearl strings (also called bubble strings) are a series of small, evenly-sized bubbles twisted into a 260 balloon, creating a delicate string-of-pearls effect. They add texture and elegance to stacks, garlands, numbers, and other displays. The tutorial covers three methods: the standard hand-held technique, a beginner-friendly "third hand" trick (tying to the display first), and how to join multiple strings for extra length.

**Key selling points for balloon professionals:**
- Transforms simple displays into premium-looking designs with minimal extra material
- Uses only a single 260 per string — extremely low cost for high visual impact
- Can be made in advance (sitting on the sofa, watching TV — Loretta's recommendation)
- Three skill levels in one technique — something for every ability
- Works with the Sempertex Silk range for an on-trend pearlescent finish

## How to Inflate a 260 for Pearls

1. **Place the 260 opening fully onto the pump nozzle** — push it all the way on, but don't cover the air outlet. If the end of the balloon covers the hole, no air comes out.
2. **Inflate with 2 pumps of air.** One pump = one full push-pull cycle (air goes in on the push AND the pull — so what looks like one arm movement is actually two pumps of air).
3. **Tie off** in a knot.
4. More pumps = more pearls per string. Two pumps gives a good standard length.

> **Pump counting note:** Loretta counts each direction of arm movement as one pump (push = 1, pull = 2). Some people count one full cycle as one pump. Clarify your counting method before following any inflation instructions.

## Method 1: Standard Hand-Held Technique

### Creating the bubbles

1. **Use two fingers as your width guide** — index and middle finger set the size of each pearl.
2. **Pinch with thumb and index finger only** — the middle finger is just the measuring spacer, not part of the pinch.
3. **Twist towards yourself, 8 full twists** — always twist the same direction, always 8 times. Consistency is what keeps bubbles locked.
4. **Move to the next pearl** — two fingers, pinch, twist 8 times. Repeat.

### Managing the string while twisting

- **After the first 2 pearls:** place the first pearl between your little finger and ring finger, pointing up and out.
- **This frees your whole hand** to keep twisting while the completed pearls hang below.
- **Never let go of the first pearl and the last pearl** — everything in between stays locked as long as the ends are held.
- **All other pearls can hang freely** — don't worry about them, they won't untwist.

### Finishing

1. **Tie off the last pearl** with a knot.
2. You now have a string held at both ends (first pearl between your fingers, last pearl knotted).

### Key reminder
> Relax your shoulders. Pearl-making causes tension — conscious relaxation produces better, more even results.

## Method 2: The "Third Hand" Trick (Beginner-Friendly)

This method is ideal for beginners or when attaching directly to a display.

### Steps

1. **Inflate the 260 with 2 pumps** and tie off as normal.
2. **Tie the knotted end directly into your display** (e.g. wrap around a 5" gap filler on a stack) before making any pearls.
3. **The display now acts as your "hand"** — it holds the first end so you don't have to.
4. **Twist pearls as normal** (two fingers, pinch, 8 twists towards yourself).
5. **You can let go between pearls** — the display holds the tension, so the string won't unravel.
6. **When finished:** tie off the last pearl, then tie the tail end into the next attachment point on your display.

### Why this works
- You effectively have "three hands" — the display holds one end, leaving both your hands free.
- Great for practising consistent pearl sizes without the stress of holding everything.
- Faster for stack decoration since you're building directly onto the display.

## Method 3: Joining Strings for Extra Length

For large displays (e.g. wrapping around a 36" or 40" number), a single 260 string won't be long enough.

### Steps

1. **Make your first string of pearls** using Method 1 (hand-held).
2. **Secure both ends temporarily:** wrap the first and last pearl around each other (like a cluster twist) so nothing unravels. Set it down.
3. **Make a second string** in a different colour if desired (e.g. Silk Pink Blossom after Silk Amethyst).
4. **Secure the second string the same way** — wrap ends around each other.
5. **Join the two strings:** take the knotted end of one string and the tail end of the other. Tie them together as tightly as possible.
6. **Cut the excess tails** close to the knot — but not so close that you risk popping a bubble.
7. **Unwrap the temporary holds** on both strings.
8. **You now have one extra-long pearl string** — ready to wrap around numbers, thick garlands, or large displays.

### Tip
- The join point will have a small knot. Position it at the back of the display where it's hidden.
- You can join as many strings as needed — repeat the process for very long runs.

## Attaching Pearl Strings to Displays

### Bottom attachment
1. Take the long tail of 260 (below the first pearl).
2. Wrap it around a 5" gap filler balloon on your display.
3. Tie off securely — this acts as an anchor.
4. Cut excess tail close to the knot.

### Top attachment
1. Take a neck from a balloon near your top attachment point.
2. Wrap the pearl string's knotted end around it.
3. Wrap once for safety first (so you don't accidentally let go), then tie off.
4. Cut excess.

### Positioning
- Pearls can drape underneath, spiral around, or lay across the display surface.
- Use glue dots to hold individual pearls in a specific position if needed.
- Reposition freely before gluing — the string is flexible.

## Tips

- **Two fingers = your measuring tool** — consistency comes from always using the same finger width
- **Always twist 8 times, always the same direction** — fewer twists and bubbles unravel; inconsistent direction causes tangles
- **Pinch with thumb and index only** — the middle finger measures, it doesn't pinch
- **Relax your shoulders** — tension in your body transfers to uneven pearls
- **Never let go of the first and last pearl** — everything else stays locked automatically
- **The Silk range is perfect for this technique** — the pearlescent finish genuinely looks like a string of pearls
- **Pre-make strings at home** — this is a sitting-on-the-sofa task, not an on-site task
- **Cut tails close but not too close** — popping a bubble at the join ruins the whole string

## Key Vocabulary

| Term | Meaning |
|---|---|
| Pearl / bubble | A single small twisted segment of a 260 — the individual "bead" in the string |
| Pearl string / bubble string | A full series of evenly twisted bubbles along a 260, creating a decorative strand |
| 260 | A long, thin modelling balloon (2" diameter, 60" length when fully inflated) |
| Pump count | The number of air pushes used to inflate — Loretta counts each direction as one pump (push = 1, pull = 2) |
| Twist lock | The principle that bubbles stay in place as long as both ends of the string are held — releasing an end unravels everything |
| Third hand trick | Tying one end of the 260 to the display before twisting, so the display holds the tension |
| Gap filler | A small (typically 5") balloon used to fill spaces in a display — also serves as an anchor point for pearl strings |
| Silk range | Sempertex's pearlescent/shimmer finish latex line — ideal for pearl string aesthetics |
| Glue dot | An adhesive dot used to position individual pearls on a display surface |
| Join | The knot where two pearl strings are tied together to create extra length |
