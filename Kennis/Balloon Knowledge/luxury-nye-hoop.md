A luxury hoop design that uses Link-o-Loon chains in a triangle-spiral pattern to cover a thick hoop frame, then adds organic balloon clusters and mini arch elements for depth and movement, finished with large statement foil balloons. ## Key Technique: Link-o-Loon Triangle Covering

This is not Alberto's original technique (he credits learning it), but it's extremely effective for covering thick frames that 260s or 360s can't wrap around.

### Why Link-o-Loons for frame covering?

Standard 260s and 360s can cover thin poles and stands. But thick frames (like large hoop frames) are too wide — you can't wrap around them. Link-o-Loon chains run alongside the frame and connect in a pattern that covers the surface without needing to wrap.

### The triangle pattern

Uses **4 Link-o-Loon chains**: 3 gold + 1 black (or any 3+1 colour combination).

#### Setup

1. **Inflate all Link-o-Loons** in the chains to approximately 4.5" (can go slightly larger depending on frame thickness). These are 6" Link-o-Loons undersized.
2. **Arrange the 4 chains** — the 3 gold chains run parallel, the 1 black chain spirals around them.

#### Starting the pattern

1. **Skip the first Link-o-Loon** in each chain — you'll need it later to close the triangle at the frame.
2. **Tie the second Link-o-Loon of each gold chain together** — this is your starting point.
3. **Include the frame inside the triangle** — the first free Link-o-Loon from each chain wraps around to close the triangle with the frame passing through the centre.

#### Building the covering

1. **The 3 gold chains form the sides of a triangular tube** running along the frame.
2. **The black chain spirals around the 3 gold chains** — wrapping and twisting around each gold line in sequence.
3. **Work methodically** — don't skip any balloon in the sequence. Move from one connection to the next.
4. **The frame sits inside the triangular structure** — completely hidden.

#### Closing

1. **At the end**, tie the final Link-o-Loons together, closing the triangle pattern.
2. **Cut any excess.**
3. The frame is now fully covered in a textured, geometric Link-o-Loon pattern.

### Variations

| Frame type | Pattern |
|---|---|
| Standard hoop (medium thickness) | Triangle (3+1 chains) |
| Larger/thicker frame | Square (4+1 chains) — add one more parallel chain |
| Tree trunk prop | Triangle or square depending on trunk diameter |

### Size options
- **6" Link-o-Loons inflated to 4.5"** — standard coverage
- **12" Link-o-Loons** — for massive frames or dramatic visual impact ("it becomes really good, really big")

## Organic Clusters (Alberto's Method)

Alberto's approach to organic clusters has a specific workflow that prioritises repositionability.

### His technique

1. **Use clusters of 3 balloons** (not 4) — tied together in different sizes.
2. **Use 12" balloons inflated to varying sizes** — intentionally inconsistent for the organic look.
3. **Tie each cluster to the frame using the balloon neck only** — wrap the neck around the frame loosely.
4. **Do NOT tie tightly at first** — leave clusters slidable so you can reposition them.
5. **Find the final position**, then secure.

### The design philosophy

> "The organic has not a real technique to do it." — Alberto

The organic section is deliberately free-form, but Alberto's specific design intent here is to create a transition where the **precise geometric Link-o-Loon pattern merges into the organic clusters** — the structured part "goes inside" the organic, blending the two styles.

## Mini Arch Elements

Small semicircular arch pieces that add detail and movement around the hoop.

### Construction

1. **Use pre-made 4-balloon clusters** (not duplets — Alberto prefers full clusters for this application).
2. **Inflate Sempertex 5" balloons to ~2.5" and ~3"** — two different sizes per cluster.
3. **The size difference creates the curve** — same principle as the Link-o-Loon football player (see: `link-o-loon-football-player.md`). Bigger difference = more curve. Here, only 0.5" difference = gentle, open curve.
4. **Connect clusters using the neck** (not fishing line) — this allows movement and repositioning.
5. **These are not organic** — all balloons must be consistently sized within their role. But they give an organic effect.

### Attaching to the hoop

1. **Tie one end** of the mini arch to the frame using an uninflated 260.
2. **Thread a 360 through the clusters** of the arch to reach the other attachment point.
3. **Pull tight** and wrap around the frame at the second point.
4. **Wrap around the frame for extra stability** — the frame provides a strong anchor.
5. **No knots needed in some cases** — friction from wrapping is sufficient for lighter elements.

## Foil Balloon Finishing

### Foil types used
- **Large champagne bottle foil** — multi-part balloon with two valves (pre-glued together in the pack, inflate each section separately).
- **Star foil** — positioned at the top or accent position.

### Alberto's foil attachment method (loop technique)

Same technique documented in `baby-balloon-column.md`:

1. **Create a loop with an uninflated 260.**
2. **Pass the foil's valve through the loop.**
3. **Tie the valve and 260 together** as a normal knot.
4. **Use the 260 tail** to tie the foil to the hoop frame or clusters.
5. **Multiple attachment points** for stability — tie at the valve, then find 1–2 more anchor points on the frame.

> "I don't wrap around the valve. I do just a loop and pass the valve through." — Alberto's consistent method across all designs.

## Tips

- **Link-o-Loons solve thick frames** — if you can't wrap a 260 or 360 around it, use the triangle chain technique
- **Don't skip balloons in the chain sequence** — skipping creates gaps in the coverage
- **Tie organic clusters loosely first** — slide them into position before securing. This is Alberto's key workflow tip
- **The 3+1 chain pattern is the standard** — upgrade to 4+1 (square) only for significantly thicker frames
- **Mini arch size difference controls curve** — 0.5" difference = gentle curve. More difference = tighter curve
- **Use necks (not fishing line) for organic clusters** — allows repositioning during the build
- **Foil loop attachment, never wrap** — the loop method is more secure and cleaner than wrapping around the valve
- **Preparation takes longer than assembly** — inflate all chains before starting the covering; the actual pattern work is fast

## Key Vocabulary

| Term | Meaning |
|---|---|
| Triangle pattern | A frame covering technique using 3 parallel Link-o-Loon chains with 1 chain spiralling around them, forming a triangular tube around the frame |
| Square pattern | The 4+1 chain variation for thicker frames — 4 parallel chains + 1 spiral |
| Link-o-Loon | A latex balloon with tails on both ends, designed to link into chains |
| Organic cluster | A group of balloons in varied sizes tied loosely to create a natural, free-form look |
| Mini arch | A small semicircular element made from consistently-sized clusters, using size differential to create a curve |
| Loop attachment | Alberto's foil connection method — passing the valve through a 260 loop before tying (not wrapping around the valve) |
| Frame covering | The process of hiding a metal frame behind balloons — Link-o-Loons for thick frames, 260/360 wraps for thin frames |
| Repositionable tie | Tying a cluster loosely using the balloon neck so it can be slid along the frame before final securing |
| Size differential curve | The principle that two parallel balloon lines of different sizes create a natural curve — used in mini arches |
