# Brontaal Analyse: Nederlandse tekst lezen voor je vertaalt

Voor je ook maar één woord vertaalt, lees de brontekst als een tolk. Wat is het register? Is het Belgisch of Nederlands? Spreektaal of geschreven taal? Formeel of informeel? Zakelijk of persoonlijk? Ironisch, dwingend, warm, koel?

Deze analyse bepaalt 80% van je vertaalkeuzes. Een tekst met "ge weet wel" mag nooit als "you are aware" vertaald worden, en "U gelieve...." mag nooit "yeah just..." worden.

## 1. Vlaams of Hollands? (BN vs NN)

### Sterke markers Belgisch Nederlands (BN):

| Marker | Voorbeeld |
|--------|-----------|
| ge/gij/u (als 2e persoon enkelvoud informeel) | "ge moet eens komen" |
| ne/nen (onbepaald lidwoord m.) | "ne keer, nen boom" |
| Diminutief op -ke | "een tasje koffie" wordt "een tasske koffie" of "een tas koffie" |
| Plaatsing "voor" achter werkwoord | "om dat te doen" wordt "voor dat te doen" |
| "Goesting" | wens, zin in iets |
| "Seffens" | dadelijk |
| "Amai" | uitroep van verbazing |
| "Allez" | als interjectie ("allez, komaan") |
| "Schoon" als "mooi" | "ne schone auto" |
| "Plezant" als "leuk" | "het was plezant" |
| "Curieus" als "nieuwsgierig" | "ik ben curieus" |
| Dubbele negatie informeel | "ik heb geen geen tijd" (zeldzaam, dialect) |
| "Tof" (gedeeld met NN, maar frequenter in BN) | |
| "Da's" of "'t Is" als zinsopener | "Da's nu net het probleem" |
| Werkwoordsvolgorde bijzin: PV achteraan | "...dat hij het gedaan heeft" of "...dat hij gedaan heeft" beide BN, "...dat hij heeft gedaan" eerder NN |

### Sterke markers Nederlands Nederlands (NN):

| Marker | Voorbeeld |
|--------|-----------|
| jij/je (informeel) | "jij moet eens komen" |
| Diminutief op -je / -tje | "kopje, autootje" |
| "Lekker" als versterker | "lekker werken, lekker thuis" |
| "Gezellig" (concept iconisch NN) | |
| "Hartstikke" als versterker | "hartstikke leuk" |
| "Joh" als afsluiter | "kom op joh" |
| Anglicismen sneller geaccepteerd | "deleten, downloaden, gefocust" |
| Werkwoordsvolgorde bijzin: PV vooraan | "...dat hij heeft gedaan" |
| Stadium voltooid deelwoord + zijn | "ik ben begonnen" (BN soms "ik heb begonnen") |
| "Even" als modaal | "ik moet even bellen" (in BN: "ik moet eens bellen") |

### Gedeeld AN (Standaardnederlands), neutraal:
Tekst zonder bovenstaande markers, met je/jij/u, normale woordvolgorde, geen regionale woordenschat. Dit is wat in handboeken staat.

## 2. Register: 5 niveaus

### Niveau 1: Hyperformeel / juridisch / officieel
- "Geachte heer/mevrouw"
- "Gelieve...."
- "Hoogachtend"
- "Bij dezen deel ik u mede dat..."
- Volledige zinnen, complexe bijzinconstructies
- Geen contracties
- Passieve constructies courant
- Nominale stijl ("de uitvoering van" ipv "het uitvoeren")
- Voorbeeld: ambtelijke brief, contract, jaarverslag, juridische tekst

### Niveau 2: Formeel zakelijk
- "Beste klant", "Beste collega"
- Volledige zinnen maar minder zwaar
- Eerder actieve dan passieve stijl
- Minimaal modale partikels
- u-vorm in BE, je-vorm of u-vorm in NL afhankelijk van bedrijfscultuur
- Voorbeeld: zakelijke mail, professionele LinkedIn post, persbericht, B2B copy voor traditionele sector

### Niveau 3: Neutraal AN
- Standaardtaal, geen sterke regionale markers
- je/jij/u-vorm
- Normale zinslengte, mix actief/passief
- Modale partikels gematigd ("wel, toch, eens")
- Voorbeeld: nieuwsartikel, professionele blogpost, neutrale marketing copy, productbeschrijving

### Niveau 4: Informeel / spreektaal
- jij/je in NN, ge/gij in BN, of mix
- Contracties: "da's, 't is, 'k heb"
- Korte zinnen, zinsfragmenten
- Modale partikels frequent: "toch, wel, eens, maar, hoor, hè, zeg"
- Stopwoorden: "eigenlijk, gewoon, zeg maar, weet je"
- Diminutieven affectief
- Anglicismen
- Voorbeeld: social media post, podcast script, casual marketing copy, interne chat, vlog, WhatsApp

### Niveau 5: Plat informeel / dialect / slang
- Sterke regionale of dialect markers
- Krachttermen, vloeken
- "Da's nen echte, hé"
- Klankweergaves: "neeje, jaa, awel"
- Voorbeeld: persoonlijke story, comedy script, character dialoog, authentieke fan content

## 3. Toon detectie

Register is hoe formeel. Toon is hoe het voelt. Dezelfde register kan verschillende tonen hebben.

### Tonen om actief naar te zoeken:

- **Zakelijk neutraal**: feitelijk, geen emotie
- **Warm/empathisch**: "ik snap het", veel "we", inclusief taalgebruik
- **Dwingend/urgent**: imperatieven, korte zinnen, "moet, nu, vandaag, snel"
- **Enthousiast/sales**: superlatieven, uitroeptekens (spaarzaam in NL maar wel marker), "ongelooflijk, geweldig, top"
- **Ironisch/sarcastisch**: omkeringen, understatement ("ja hoor", "tof zeg"), conditioneel
- **Educatief/uitleggend**: "stel je voor dat", "het zit zo", "dit betekent dat"
- **Confronterend/kritisch**: directe vragen, "waarom", uitdagende statements
- **Verhalend**: chronologische verbinders, scenes, dialoog, sensorische details
- **Reflectief/filosofisch**: open vragen, voorwaardelijk, "misschien, eigenlijk, in zekere zin"
- **Speels/humoristisch**: woordspelingen, overdrijvingen, knipogen

### Modale partikels lezen (cruciaal voor toon):

| Partikel | Functie | Voorbeeld |
|----------|---------|-----------|
| wel | nuancering, geruststelling | "het komt wel goed" |
| toch | verwachting/contrast | "je komt toch?" |
| eens | uitnodigend, vriendelijk | "kom eens kijken" |
| maar | versoepelend | "doe maar" |
| hoor | geruststellend (NN sterker) | "het is oké hoor" |
| hè | bevestigend, verbindend | "leuk hè" |
| zeg | verbazing, contact | "leuk zeg" |
| nou | herinnering, lichte irritatie | "nou ja" |
| even | minimaliserend | "even kijken" |
| gewoon | banaal makend | "het is gewoon dom" |
| eigenlijk | herziening | "eigenlijk wel" |

Deze partikels HEBBEN equivalenten in elke doeltaal, maar nooit letterlijk. Soms zijn ze een zinsbouwkeuze, soms een ander woord, soms een intonatieaanwijzing die in geschreven taal verloren gaat.

## 4. Stijlkenmerken

- **Zinslengte**: korte zinnen suggereren punch, ritme, urgentie. Lange zinnen suggereren reflectie, complexiteit, autoriteit.
- **Zinsfragmenten** ("Klaar. Punt. Echt."): retorisch, marketing, copy, persoonlijke stijl
- **Vraagzinnen**: dialoog met lezer, retorische functie, of echte vraag
- **Actief vs passief**: actief = directe stijl, passief = afstand of focus op handeling
- **Naamwoordstijl vs werkwoordstijl**: "de uitvoering van het project" (naamwoord, zakelijk) vs "het project uitvoeren" (werkwoord, dynamisch)
- **Anglicismen**: hoge dichtheid suggereert tech, marketing, jonge doelgroep
- **Vakjargon**: dichtheid en type bepaalt doelpubliek

## 5. Doel van de tekst

Probeer altijd te identificeren:

- **Informeren**: feiten, neutraal, structureel duidelijk
- **Overtuigen**: argumentatie, sociale bewijzen, call to action
- **Verkopen**: hook, pijn, oplossing, urgentie, prijs
- **Onderwijzen**: stap voor stap, voorbeelden, samenvatting
- **Vermaken**: ritme, verrassing, emotie, karakter
- **Verbinden**: warmte, persoonlijkheid, gedeelde ervaring
- **Mobiliseren**: dwingend, collectief, missie

Het doel kan de vertaling sturen: een sales tekst moet in de doeltaal nog steeds verkopen, dus de vertaling moet de sales kracht overnemen, niet alleen de woorden.

## 6. Doelpubliek

- **Algemeen publiek**: vermijd jargon, eenvoudige zinsbouw, voor de hand liggende referenties
- **Specialisten**: jargon mag, korte zinnen kunnen ok zijn want lezers vullen aan
- **Internationaal**: vermijd cultuurspecifieke referenties
- **Specifiek nationaal**: lokale referenties mogen

## 7. Checklist voor je vertaalt

Beantwoord deze vragen voor jezelf voor je begint:

- [ ] Is dit BN, NN, of neutraal AN?
- [ ] Welk register (niveau 1 tot 5)?
- [ ] Welke dominante toon?
- [ ] Welke modale partikels staan erin en wat doen ze?
- [ ] Wat is het doel van de tekst?
- [ ] Wie is het doelpubliek?
- [ ] Wat is het medium (mail, post, advertentie, video script, etc.)?
- [ ] Zitten er idiomen/uitdrukkingen in die equivalenten nodig hebben?
- [ ] Zitten er cultuurspecifieke referenties in?
- [ ] Zijn er marketingtermen of vakjargon die in het doeltaal vakgebied anders heten?

Pas DAARNA open je het doeltaal bestand.

## 8. Veelvoorkomende valkuilen in de analyse

- **De auteur schrijft niet zoals hij/zij spreekt**: zakelijke ondernemer kan in mails formeel zijn, in WhatsApp losse spreektaal. Beoordeel deze specifieke tekst, niet de persoon.
- **Vlaams hoeft niet dialect te zijn**: een Vlaamse zakelijke mail kan perfect AN zijn. Niet elke BN tekst is ge/gij.
- **Anglicismen ≠ informeel**: tech sector tekst kan zwaar Engels gekleurd zijn en toch zakelijk.
- **Ironie wordt vaak gemist**: zoek naar overdrijving, contradictie tussen woord en context, understatement.
- **Korte zin ≠ informeel**: copywriting heeft korte zinnen voor effect, niet voor informaliteit.

Alleen na deze analyse open je het doeltaal bestand.
